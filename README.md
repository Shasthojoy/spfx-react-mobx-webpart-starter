# spfx-react-mobx-webpart-starter

A starter project for a [React](https://github.com/facebook/react) & [Mobx](https://github.com/mobxjs/mobx) powered [spfx](https://github.com/SharePoint/sp-dev-docs) webpart.

```
npm install
npm start
```
